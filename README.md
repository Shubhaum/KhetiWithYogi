# KhetiWithYogi

Static GitHub Pages website for KhetiWithYogi, a net house farming consultancy serving Ajmer, Pushkar, Kishangarh and nearby Rajasthan farming areas.

## Pages

- `index.html` - Home page
- `services.html` - Consultancy services
- `crops.html` - Crop expertise
- `about.html` - About the consultant
- `contact.html` - Enquiry page

## Before Publishing

1. Open `assets/js/main.js`.
2. Replace `CLIENT_WHATSAPP_NUMBER = ""` with the client WhatsApp number in international format without `+`, for example `919876543210`.
3. Update contact copy if you want to add a public phone number, email or farm address.
4. If using a custom domain, add the domain in GitHub Pages settings and create a `CNAME` file containing the domain.

## GitHub Pages Hosting

1. Push this folder to a GitHub repository.
2. In GitHub, open repository `Settings`.
3. Go to `Pages`.
4. Under `Build and deployment`, choose `Deploy from a branch`.
5. Select the main branch and root folder.
6. Save and wait for the GitHub Pages URL.

No build command is required.
